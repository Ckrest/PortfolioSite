Portfolio Website Refresh

Launched on October 1, 2025, this rebuild focuses on delivering a lightweight, framework-free experience with an automated project pipeline. Every card on the homepage is driven by structured data, making future updates frictionless.

Design System Frontend Content Strategy

What changed

The refresh ships a cohesive typographic scale, new color tokens, and a responsive layout tuned for both desktop and mobile storytelling. Inline CSS and JavaScript keep the bundle tiny while still enabling interactive sections like the featured project callout.

Projects now live in a single JSON file, letting the build process and the live site share the same source of truth. The detail pages inherit a consistent template and accessibility improvements, including explicit alt text and keyboard-friendly navigation.

Impact

  • Sub-30KB initial payload and perfect Lighthouse performance scores.
  • CMS-free publishing flow: update projects.json and drop in a new detail page.
  • Improved SEO metadata, including automated descriptions and structured dates.
